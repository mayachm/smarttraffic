# SmartTraffic RL Project

This project trains and tests a traffic-light RL agent using SUMO.

## Requirements

- Python 3.9+
- SUMO 1.24+ installed and added to PATH
- Packages listed in `requirements.txt`

## Setup

```bash
# Create environment
conda create -n smarttraffic python=3.9
conda activate smarttraffic

# Install packages
pip install -r requirements.txt
